幼い頃、1度や2度、本や映画の主人公になりたいと思ったことはありませんか。

主人公になってあんな冒険をしてみたり、こんなできごとに遭遇したり。

このものがたりを体験することでだれもが「自分が主役」の絵本をつくることができます。

つくった絵本はスマートフォンにいれて持ちかえることだってできます。

画面の前に立って手や足を動かします。

飛んでくる葉っぱや、降ってくる雨に当たらないように、傘を体で守りましょう。

ゲームが終わると自分だけの絵本ができあがります。

当たった回数や、行った動作によってできあがるストーリーが変わります。

遊んだ画像をQRコードからダウンロードすることで持ちかえることができます。

本当の絵本として、家で読むこともできるようになっています。

大切な人に見せてあげたいものです。
